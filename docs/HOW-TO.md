## â–¶ Installation
### Windows
1. Download Python from [python.org](https://www.python.org/downloads/)
2. Check **"Add Python to PATH"** during installation
3. In Command Prompt :
```cmd
cd Downloads/ChefAI
pip install -r requirements.txt
python chefai.py
```

### MacOS/Linux
1. Open Terminal
2. Paste these commands :
```bash
cd ~/Downloads/ChefAI
pip3 install -r requirements.txt
python3 chefai.py
```
